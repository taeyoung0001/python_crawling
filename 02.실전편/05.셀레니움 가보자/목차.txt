# 목차

1. 셀레니움 환경 설정
    - 셀레니움 라이브러리 설치
    - 웹드라이버 임포트
    - 웹드라이버 객체 생성

2. 웹 브라우저 기능 조작 한방에 끝내기
    - 원하는 페이지로 이동 방법
    - 브라우저 내 이동: 뒤로 가기, 앞으로 가기
    - 페이지 새로고침
    - 브라우저 세션 종료
    - 현재 URL 확인
    - 페이지 제목 확인
    - 창 크기 조절 : 최대화, 최소화

3. 원하는 태그 찾고 제어하기 
- css 선택자로 태그 찾기
- 태그 클릭하기
- 태그 입력하기
- 키보드 키 입력하기 (ex.enter)
- 셀레니움 기본 템플릿 정리

4. 셀레니움 실전 테크닉

    1. 여러 태그 동시 제어하기
        -여러 태그 찾기 및 클릭하기

    2. 동적 대기 기법
        - 암시적 대기
        - 명시적 대기

    3. 고급 입력 컨트롤
        - 입력 값 삭제
        - 순차적 키 입력
        
    4. 셀렉트박스 조작 방법
        - select 클래스를 이용한 드롭다운 메뉴 선택
        
    5. 여러 페이지 관리
        - 페이지 안에 다른 페이지가 있는 경우 (iframe 제어 방법)
        - 새로운 창 제어 방법
