# 네이버 메일 자동 완성화 코드

# 목표 : 네이버에 로그인해서 메일쓰기를 누른 후 자동으로 메일을 보내보기

# 문서화
1. 아이디랑 비밀번호 값 변수로 지정
2. 네이버 페이지로 들어가서 로그인 창에 아디, 비번 입력
3. 메일 카테고리 접속 -> 새창으로 열리는 거 확인하고 스위칭 해줌
    driver.switch_to.window(driver.window_handles[1])
4. 본문쓰기는 아이프레임으로 되어 있으므로, 아이프레임을 활용하여 본문에 작성
5. 타이틀이랑, 본문쓰기 제대로 된거 확인하고 전송 보내면 끝~
