# pySide 는 상업적 이용 시 라이센스 필요 없음

# 설치방법
    pip install pyside6 로 설치함

    그리고 Requirement  에 뜨는 주소들 복사!
    탐색기에 가서 그 주소들 붙여넣어줌

    그러면 pyside6 찾고 designer.exe 파일 실행 후
    위젯으로 설저하고 start~

# pyside6 사용법
    모눈종이같은데에 위젯을 붙여서 사용할 거임
    맥사용자들은 설정들어가서 사용자인터페이스에서 도킹 된 창으로 변경하셈
    
    알아서 잘 편집하고 작업할 폴더에 저장하셈
    pyside6-uic login.ui -o login.py 로 파일 명은 py로 해야함
   
    💫 주의사항 💫

    ui 파일을 파이사이드로 편집하면 실시간으로py에 동기화 되지 않았다
    그래서  pyside6-uic 이전 유아이.ui -o 바꿀 유아이.py 다시 하니깐 동기화가 잘 되었다.

# .exe 파일로 만드는 법
    1. pip install pyinstaller 
    2. pyinstaller -w -F 03.pyside6_네이버지식인.py  작업할 경로로 가야함 
    pyinstaller : 이 라이브러리 사용할거임
    -w : 기본적인 콘솔창을 뜨지 않게 한다
    -F : 하나의 exe 파일만 사용하게 할 수 있다(필수)
    03.pyside6_네이버지식인.py : 파일 경로 맞추고 작업한 파일 쓰셈
