# int와 float의 차이점 정리해보기

# int (정수형)
    정의: 정수를 나타내는 자료형.
    값의 범위: 소수점이 없는 숫자 (예: 1, -5, 0)

    특징:
        더 빠르고 메모리 사용량이 적음.
        소수점 이하의 값이 필요하지 않은 경우 사용.

# float (부동소수점형)
    정의: 실수를 나타내는 자료형.
    값의 범위: 소수점 이하 값을 포함 (예: 1.5, -3.14, 0.0)

    특징:
        더 많은 메모리를 사용하며, 소수점 이하를 처리할 수 있음.
        계산 시 약간의 부정확성이 발생할 수 있음 (부동소수점 연산의 특성상).
